# 12 个月西班牙语启蒙计划

## 总体原则

- 每个场景每次只引入 3-5 句新句子，反复使用至少一周再加新的
- 所有阶段的句子都会持续使用，阶段只是"新增"而非"替换"
- 家长对孩子全程只说西班牙语，中文翻译仅供家长参考
- 不强迫孩子开口，先输入后输出，耐心等待

---

## 阶段 1：听懂并执行（第 1-2 月）

### 目标
- 孩子能听懂简单指令并用动作回应
- 能用 sí / no 或单词回答简单问句
- 在日常流程中建立"听到西语→做出反应"的习惯

### 核心句型

| 句型 | 示例 | 用途 |
|------|------|------|
| 祈使句 | Ven aquí. / Siéntate. / Dame el zapato. | 直接指令 |
| Vamos a + 动词原形 | Vamos a comer. / Vamos a dormir. | 宣布活动 |
| ¿Quieres + 名词? | ¿Quieres agua? / ¿Quieres más? | 是/否问句 |
| ¿Dónde está + 名词? | ¿Dónde está tu oso? | 寻找游戏 |
| Mira + 名词 | Mira el perro. / Mira, un avión. | 引起注意 |
| Toma / Ten | Toma tu leche. / Ten el libro. | 递给东西 |

### 第 1 月重点
- **第 1-2 周**：卫生间 + 餐桌场景（每天最高频的场景）
  - 重点句：Vamos al baño / Lávate las manos / Siéntate / ¿Quieres más?
  - 目标：孩子听到能执行，不需要翻译
- **第 3-4 周**：加入卧室 + 大门玄关
  - 重点句：Vamos a dormir / Ponte los zapatos / Vamos / ¿Dónde está tu abrigo?
  - 复习前两周句子

### 第 2 月重点
- **第 5-6 周**：加入客厅 + 汽车内
  - 重点句：Siéntate aquí / Ven aquí / Ponte el cinturón / Mira por la ventana
  - 积累场景转换句：Vamos a la sala / Vamos al coche
- **第 7-8 周**：覆盖全部场景（电梯、车库、幼儿园门口、厨房）
  - 重点句：Espera / Cuidado / Dame la mano / Di adiós
  - 全面巩固所有阶段 1 句子

### 家长策略
- 每次说指令时配合手势和动作演示
- 利用"¿Dónde está...?"做寻找游戏，激发兴趣
- 孩子执行后立即用"¡Muy bien!"表扬
- 不要求孩子重复，但如果孩子模仿要热情回应

## 阶段 2：短句输出（第 3-4 月）

### 目标
- 孩子开始产出 2-3 词的短句
- 能用"Yo quiero + 名词"表达需求
- 能用简单形容词描述事物
- 能在选择问句中做出回答

### 核心句型

| 句型 | 示例 | 用途 |
|------|------|------|
| Yo quiero + 名词 | Yo quiero agua. / Yo quiero más. | 表达需求 |
| Es + 形容词 | Es grande. / Es bonito. | 描述事物 |
| Está + 状态 | Está caliente. / Está frío. | 描述状态 |
| Me gusta / No me gusta | Me gusta el perro. | 表达喜好 |
| 选择问句 | ¿Quieres agua o leche? | 引导选择 |
| Tengo + 名词 | Tengo hambre. / Tengo sueño. | 表达身体状态 |
| Mío/Tuyo | Es mío. / Es tuyo. | 表达所属 |

### 第 3 月重点
- **第 9-10 周**：在餐桌和卫生间引入选择问句
  - 家长问：¿Quieres pan o fruta? 孩子答：Pan. → 家长示范：Yo quiero pan.
  - 引入温度/味道词汇：caliente, frío, rico, dulce
- **第 11-12 周**：在客厅和卧室引入描述
  - 家长指着东西说：Es grande. ¿Es grande o pequeño?
  - 引入颜色复习：Es rojo. ¿De qué color es?
  - 引入"Me gusta"：Me gusta este libro. ¿Te gusta?

### 第 4 月重点
- **第 13-14 周**：鼓励"Yo quiero"主动表达
  - 当孩子指着东西时，引导说：Di "Yo quiero..."
  - 扩展到所有场景：Yo quiero salir / Yo quiero jugar / Yo quiero dormir
- **第 15-16 周**：引入 Tengo + 身体状态
  - Tengo hambre / Tengo sed / Tengo sueño / Tengo frío / Tengo calor
  - 家长用夸张表情示范，鼓励孩子模仿

### 家长策略
- 大量使用选择问句（A o B?）降低输出难度
- 孩子说单词时，家长补全为完整短句（"水" → "Sí, yo quiero agua"）
- 用实物对比教形容词（大球 vs 小球：grande / pequeño）
- 每天至少 3 次"Me gusta / No me gusta"的示范

## 阶段 3：简单对话（第 5-6 月）

### 目标
- 孩子能用简单句回答开放式问题
- 能描述正在做的事
- 能进行 2-3 轮的简单对话
- 开始使用"porque"给出简单原因

### 核心句型

| 句型 | 示例 | 用途 |
|------|------|------|
| ¿Qué estás haciendo? | — Estoy comiendo. | 描述进行中的动作 |
| ¿Qué es esto? | — Es un coche. | 识别事物 |
| ¿Por qué? / Porque... | ¿Por qué? — Porque tengo hambre. | 简单因果 |
| ¿Puedo + 动词? | ¿Puedo jugar? / ¿Puedo salir? | 请求许可 |
| Primero... después... | Primero manos, después comer. | 顺序 |
| Ya + 过去分词 | Ya terminé. / Ya está. | 完成 |
| ¿De quién es? | Es de Papá. / Es mío. | 所属 |

### 家长策略
- 大量自言自语描述自己的动作（Estoy abriendo la puerta...）
- 不急于要求完美句子，接受"关键词"回答
- 用"Primero... después..."建立日常流程的可预期性
- 对孩子的"¿Por qué?"保持耐心，用简单西语回答
- 开始在对话中融入 Abuelo/Abuela 的角色

---

## 阶段 4：描述与叙述（第 7-8 月）

### 目标
- 孩子开始使用简单过去时描述发生过的事
- 能用方位词描述东西在哪里
- 能数数和表达数量
- 能用更丰富的词汇描述事物

### 核心句型

| 句型 | 示例 | 用途 |
|------|------|------|
| 简单过去时 | Comí arroz. / Jugué en el parque. / Fui al parque. | 描述过去的事 |
| ¿Qué hiciste? | — Pinté un dibujo. | 询问过去的事 |
| 方位介词 | Está encima de / debajo de / al lado de / dentro de | 描述位置 |
| Hay + 数量 | Hay dos gatos. / Hay muchos coches. | 表达数量 |
| ¿Cuántos? | ¿Cuántos hay? — Tres. | 询问数量 |
| Vi / Oí | Vi un perro. / Oí un ruido. | 描述感知 |

### 第 7 月重点
- **第 25-26 周**：引入简单过去时（高频动词）
  - 只教最常用的：comí, bebí, jugué, dormí, fui, vi, hice
  - 在接孩子放学时大量练习：¿Qué hiciste hoy? ¿Qué comiste?
- **第 27-28 周**：引入方位介词
  - 用寻找游戏练习：¿Dónde está el oso? — Está debajo de la mesa.
  - 每次只教一个方位词，掌握后再加下一个

### 第 8 月重点
- **第 29-30 周**：引入数量表达
  - 日常中数东西：¿Cuántos dedos? ¿Cuántas galletas quieres?
  - 配合数字复习：uno, dos, tres, cuatro, cinco
- **第 31-32 周**：综合叙述练习
  - 鼓励孩子复述简单事件：Hoy fui al parque. Vi un perro.
  - 家长先示范，孩子模仿

### 家长策略
- 每天接孩子放学后问"¿Qué hiciste hoy?"作为固定环节
- 过去时只教不规则高频动词的第一人称，不讲语法
- 方位词用"藏东西找东西"游戏来教
- 不纠正过去时的错误变位，只需自然重复正确形式

---

## 阶段 5：感受与想法（第 9-10 月）

### 目标
- 孩子能用西语表达情绪和感受
- 能用"Voy a..."表达打算
- 能进行简单比较
- 能礼貌地请求帮助

### 核心句型

| 句型 | 示例 | 用途 |
|------|------|------|
| Estoy + 情绪 | Estoy contento / triste / enfadado / cansado / asustado | 表达情绪 |
| Voy a + 动词 | Voy a jugar. / Voy a dormir. | 表达打算 |
| Más... que | Es más grande que eso. | 比较 |
| ¿Me ayudas / Me das? | ¿Me ayudas con esto? / ¿Me das agua? | 礼貌请求 |
| Yo sé + 动词 | Yo sé hacerlo. / Yo sé vestirme. | 表达能力 |
| No sé | No sé. / No sé dónde está. | 表达不知道 |
| También / Tampoco | Yo también. / Yo tampoco. | 附和 |

### 第 9 月重点
- **第 33-34 周**：情绪词汇
  - 通过表情、绘本、日常情境教：contento, triste, enfadado, cansado, asustado
  - 每天问：¿Cómo estás? → Estoy contento.
  - 家长也主动表达自己的情绪做示范
- **第 35-36 周**：引入"Voy a..."表达打算
  - 每次做事前说：Voy a cocinar. / Voy a salir.
  - 问孩子：¿Qué vas a hacer? → Voy a jugar.

### 第 10 月重点
- **第 37-38 周**：比较和请求
  - 用实物比较大小：Este es más grande que ese.
  - 教礼貌请求：¿Me das...? / ¿Me ayudas...?
- **第 39-40 周**：自我能力表达
  - 鼓励"Yo sé + 动词"：Yo sé ponerme los zapatos solo.
  - 接受"No sé"作为合理回答

### 家长策略
- 情绪词要在真实情境中教，不要脱离场景
- 当孩子发脾气时，帮他命名情绪：Estás enfadado porque...
- "Voy a..."可以变成日常预告：Voy a apagar la luz. ¿Listo?
- 比较句用实物演示，不抽象讲解
- 每次孩子做到什么，积极说：¡Tú sabes hacerlo solo! ¡Muy bien!

## 阶段 6：自主表达（第 11-12 月）

### 目标
- 孩子能用连接词把句子串起来
- 能复述简单事件（先...然后...最后...）
- 能主动发起问题
- 开始使用简单假设句
- 能在角色扮演游戏中使用西语

### 核心句型

| 句型 | 示例 | 用途 |
|------|------|------|
| Y / Pero / También | Quiero pan y leche. / Quiero salir, pero llueve. | 连接句子 |
| Primero... luego... al final | Primero comí, luego jugué, al final dormí. | 复述事件 |
| ¿Por qué? ¿Cómo? ¿Cuándo? | ¿Cuándo vamos? / ¿Cómo se hace? | 主动提问 |
| Si + 条件 | Si llueve, no salimos. / Si como todo, ¿puedo jugar? | 简单假设 |
| Yo creo que... | Yo creo que está lejos. | 表达观点 |
| Hagamos que... | Hagamos que el sofá es un barco. | 角色扮演 |
| Cuando + 时间 | Cuando llegamos, quiero jugar. | 时间关联 |

### 第 11 月重点
- **第 41-42 周**：连接词和复述
  - 教 y, pero, también 在日常中的使用
  - 用三步法引导复述：Primero... luego... al final...
  - 每天睡前回顾一天：¿Qué hicimos hoy?
- **第 43-44 周**：主动提问
  - 鼓励孩子用 ¿Por qué? ¿Cuándo? ¿Cómo? 提问
  - 对孩子的问题用简单西语耐心回答

### 第 12 月重点
- **第 45-46 周**：假设句和角色扮演
  - 在游戏中引入：Hagamos que tú eres el doctor.
  - 简单假设：Si llueve, nos quedamos en casa.
- **第 47-48 周**：综合巩固
  - 回顾所有阶段，查漏补缺
  - 尝试让孩子独立讲一个简短的小故事
  - 庆祝一年的进步！

### 家长策略
- 这个阶段家长要"少说多听"，给孩子表达空间
- 角色扮演游戏（过家家、看医生等）是最好的语言练习
- 不追求语法完美，重点是敢说、能表达
- 睡前回顾一天用"Primero... luego... al final..."串联
- 用"Yo creo que..."示范表达观点，鼓励孩子也用

## 每日建议时间分配

不需要专门"上课"，融入日常即可：

| 场景 | 预计每日接触次数 | 备注 |
|------|------------------|------|
| 卫生间 | 4-6 次 | 洗手、刷牙、上厕所 |
| 餐桌 | 3 次 | 三餐 |
| 卧室 | 2-3 次 | 起床、午睡、睡前 |
| 客厅 | 持续 | 主要活动区域 |
| 大门玄关 | 2-4 次 | 出门/回家 |
| 厨房 | 1-2 次 | 安全提醒为主 |
| 电梯 | 2-4 次 | 出门/回家时 |
| 地下车库 | 1-2 次 | 开车出行时 |
| 汽车内 | 视情况 | 通勤/出行 |
| 幼儿园门口 | 2 次 | 送/接 |
